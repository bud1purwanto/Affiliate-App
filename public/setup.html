🔒 Setup Token

1. OpenRouter (AI) OFF

Ambil API key di openrouter.ai/keys. Tanpa ini, app tetap jalan pakai template fallback.

2. Shopee Affiliate API OFF

Daftar Open API di dashboard Shopee Affiliate → menu Open API.

3. Threads API (opsional) OFF

Hanya untuk auto-post dari Web App. Untuk posting praktis, gunakan Chrome Extension (tidak butuh ini). Token dari Threads API Meta.

4. Akun Threads — Server 0

Akun yang kamu set di Cloudflare dan bisa dipakai siapa pun yang buka web ini (mis. istri tinggal pakai). Token tersimpan aman di server (tidak ditampilkan di web).

Cara set (sekali saja):
1. Cloudflare → project → Settings → Variables and Secrets → tambah variabel THREADS_ACCOUNTS (tipe Secret), isinya JSON:
[{"label":"Budi","token":"TH_xxx","userId":"123"}, {"label":"Istri","token":"TH_yyy","userId":"456"}] 2. Retry deployment. Setelah itu akun muncul di sini & di pemilih akun saat posting.
Token long-lived & user ID tiap akun didapat dari Threads API Meta.

5. Akun Threads — Browser ini 0

Opsional. Akun yang disimpan hanya di browser ini (tidak ikut untuk pengunjung lain). Untuk dipakai bersama (istri di HP-nya), pakai akun Server di atas.

Di Railway, filesystem bersifat ephemeral — credential aktif untuk sesi ini. Untuk permanen, set juga di Railway → Variables.
